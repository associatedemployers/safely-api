## Safely API
The api for the safely app.

### Boot

* [Node.js](https://nodejs.org/) (with npm) ![Release](https://img.shields.io/static/v1?label=Node&message=10.24.1&color=4baf4f)

Production
```
$ npm start
```

Dev
```
$ npm run-script watch
```


### Tests

```
$ npm test
- OR -
$ npm run-script test-watch
```
